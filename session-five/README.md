# Beginner's Python: Session Five

## Session Content

### Standard Content

* Learn how to use `for` loops to run code a predetermined number of times
* Loop through lists and ranges of numbers
* Escape loops using `break` and `continue`

* Else
* Ignoring assignment
* List comprehension
* Matrices
* Zipping
